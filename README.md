# [Título del trabajo]

haremos una réplica de un parking público como los que podemos encontrar en las grandes ciudades. Reproduciendo un sistema de pago e identificación de matrículas, de señalización de plazas libres, y acceso mediante semáforos coordinados con los de la calle en la que se encuentre el acceso al parking. Además le daremos un toque más actual implementando un sistema de ayuda al aparcamiento conformado tanto por leds como por advertencias sonoras además de un sistema de guiado hacia las plazas libres para que los conductores no tengan que perder el tiempo buscando.  Tambien introduciremos un sistema de guiado para los usuarios que les permita llegar a una plaza libre de la forma mas rapida y comoda posible.

## Objetivos del trabajo
El objetivo es estudiar y entender las similitudes y diferencias entre arduino y visual, aprender a manejar dichos programas con una mayor soltura. Necesaria para los próximos cursos del grado así como aprender a darle solución a problemas prácticos presentes en la vida real de la manera más práctica, eficiente y sencilla posible.
